# Narendra Jadaun Portfolio

A modern React + TypeScript portfolio for Narendra Jadaun, focused on backend engineering, scalable reward platforms, AWS infrastructure, production debugging, and database optimization.

## Tech Stack

- React
- TypeScript
- Vite
- Lucide React
- CSS glassmorphism and responsive layouts

## Getting Started

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Lint:

```bash
npm run lint
```

## Deployment

This is a static Vite site. Deploy the `dist` folder after running `npm run build`.

Recommended free hosting:

- Cloudflare Pages
- Vercel
- Netlify
